# Contexto do Sistema\n\nEste sistema é feito para que alunos consigam acompanhar seu desempenho nas disciplinas que estudam.\n\nEntidades principais:\n- usuario (alunos)\n- disciplina\n\nRegras de autorização:\n- Um usuário só possui permissão de leitura/escrita em disciplinas vinculadas ao seu próprio user_id.\n- Um usuário pode ter diversas disciplinas cadastradas.\n- Cada disciplina pertence a exatamente um usuário.\n\nBanco de dados:\n- Xano\n\nFuturo previsto:\n- APIs para expor funcionalidades e dados\n- Tarefas automatizadas para cálculo de métricas, notificações e manutenção\n- Integrações com ferramentas de aprendizado, notas e autenticação externa
